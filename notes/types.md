
# Основные

| размер  | знаковый | беззнаковый | с плавающей точкой          |
| ------- | -------- | ----------- | --------------------------- |
| 8-бит   | sbyte    | byte (db)   |                             |
| 16-бит  | sword    | word        |                             |
| 32-бит  | sdword   | dword       | real4 одинарной точности    |
| 64-бит  | sqword   | qword       | real8 двойной точности      |
| 80-бит  |          | tbyte       | real10 расширенной точности |
| 128-бит |          | oword       |                             |

# Вектора для SIMD

| размер   | название | регистр |
| -------- | -------- | ------- |
| 128 байт | xmmword  | xmm     |
| 256 байт | ymmword  | ymm     |
| 512 байт | zmmword  | zmm     |
